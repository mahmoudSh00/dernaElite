# Derna Elite - متجر درنة للتسوق

## متجر إلكتروني متكامل مبني باستخدام Next.js, React, Tailwind CSS و Firebase

### الميزات
- 🛒 سلة التسوق
- 📱 دعم RTL كامل للغة العربية
- 📊 لوحة تحكم للمشرف
- 📱 منصة واتساب لاستقبال الطلبات
- 📱 تصاميم متجاوزة لأجهزة مختلفة
- 📊 تحسين SEO
- 📱 lazy loading للصور

### التثبيت

1. قم بتحميل أو استنساخ المشروع
2. قم بتثبيت التبعيات: `npm install`
3. أنشئ ملف `.env.local` واضف إعدادات Firebase فيه (ستجد نموذج في `.env.local.example`)
4. شغِّل الخادم المحلي: `npm run dev`
5. افتح المتصفح على `http://localhost:3000`

### نشر على Vercel

- اربط مستودعك بـ Vercel
- أضف المتغيرات البيئية (Environment Variables) في إعدادات المشروع على Vercel
- انشر!
