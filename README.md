# POS Pro - نظام مبيعات احترافي Offline-First

نظام مبيعات متكامل (Point of Sale) يعمل بالكامل داخل المتصفح بدون الحاجة لسيرفر (Backend-less). مصمم للعمل بكفاءة على الأجهزة الضعيفة والهواتف، ويدعم العمل بدون إنترنت (Offline).

## 🚀 المميزات الأساسية

*   **بدون سيرفر**: يعتمد كلياً على `IndexedDB` و `LocalStorage` في المتصفح.
*   **يعمل بدون إنترنت (Offline First)**: بفضل تقنية PWA و Service Workers.
*   **سريع جداً**: مكتوب بـ Vanilla JS (بدون مكتبات ضخمة).
*   **إدارة المخزون**: إضافة، تعديل، حذف منتجات، وتتبع الكميات.
*   **نقاط البيع (POS)**: واجهة كاشير سهلة تدعم الباركود والبحث السريع.
*   **إدارة المستخدمين**: صلاحيات (مدير، موظف، كاشير).
*   **تقارير**: متابعة المبيعات اليومية وتنبيهات المخزون.
*   **نسخ احتياطي**: إمكانية تصدير واستيراد قاعدة البيانات كملف JSON.

## 🛠️ التقنيات المستخدمة

*   **HTML5 & CSS3**: تصميم متجاوب (Responsive) يدعم الهواتف والوضع الليلي (Dark Mode).
*   **JavaScript (ES6+)**: المنطق الكامل للتطبيق.
*   **IndexedDB**: قاعدة بيانات المتصفح لتخزين المنتجات والعمليات.

## 📥 طريقة التشغيل

1.  حمل المجلد بالكامل.
2.  افتح ملف `index.html` في أي متصفح حديث (Chrome, Edge, Firefox, Safari).
3.  **بيانات الدخول الافتراضية**:
    *   المستخدم: `معتصم`
    *   كلمة المرور: `12121212`

## 📱 التثبيت كطبييق (PWA)

*   على **الكمبيوتر**: اضغط على أيقونة "تثبيت" (+) في شريط العنوان بالمتصفح.
*   على **Android**: افتح القائمة في Chrome واختر "Install App" أو "Add to Home screen".
*   على **iOS**: اضغط زر المشاركة في Safari واختر "Add to Home Screen".

## 📂 هيكلية المشروع

*   `index.html`: نقطة البداية.
*   `js/db.js`: التعامل مع قاعدة البيانات.
*   `js/auth.js`: نظام الدخول والصلاحيات.
*   `js/app.js`: منطق الواجهة والتحكم.
*   `css/style.css`: التنسيقات.
*   `sw.js`: ملف Service Worker لدعم العمل بدون إنترنت.
